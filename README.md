# cortex


## Installation

1. Clone the repository and enter it:

    ```sh
    $ git clone git@github.com:taufnast/cortex.git
    ...
    $ cd cortex/
    ```

2. Run the installation script and activate the virtual environment:

    ```sh
    $ ./scripts/install.sh
    ...
    $ source .env/bin/activate
    [cortex] $ # you're good to go!
    ```
